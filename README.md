# Discord Bot + Web Dashboard

Мощный Discord бот с веб-панелью управления для модерации и настройки.

## 📦 Установка

### 1. Скачайте репозиторий

### 2. Запустите настройку

```bash
python setup.py
```

Введите данные:
- Токен Discord бота
- Префикс команд
- ID Discord сервера
- Категории тикетов
- Максимальное количество предупреждений
- Логин и пароль для сайта (или оставьте пустыми для автогенерации)

### 3. Запуск

```bash
python run.py
```

или раздельно:
```bash
# Терминал 1: Бот
cd bot && python main.py

# Терминал 2: Сайт
cd website && python -m uvicorn app:app --host 0.0.0.0 --port 8000
```

## 🌐 Доступ к сайту

- Локально: `http://localhost:8000`
- Через интернет: `http://ВАШ_IP:8000`

## 🌩️ Настройка HTTPS (Cloudflare)

1. Зарегистрируйтесь на [cloudflare.com](https://cloudflare.com)
2. Добавьте домен
3. Измените NS сервера на Cloudflare
4. Добавьте DNS запись CNAME

## 🔧 Функционал сайта

| Страница | Описание |
|----------|----------|
| Dashboard | Статистика сервера |
| Команды | Управление кастомными командами |
| Приветствие | Настройка приветствия новых участников |
| Тикеты | Просмотр тикетов |
| Модерация | Бан/кик/варн пользователей по ID |
| Логи | История модерации |
| Пользователи | Список участников сервера |
| Настройки | Префикс, категории тикетов |

## 🎮 Команды бота

### Модерация
- `!ban @user [причина]` - Забанить
- `!unban ID` - Разбанить
- `!kick @user [причина]` - Кикнуть
- `!mute @user [время]` - Замутить
- `!unmute @user` - Размутить
- `!warn @user [причина]` - Предупреждение
- `!warns @user` - Список предупреждений
- `!clearwarns @user` - Очистить предупреждения
- `!clear [кол-во]` - Очистить сообщения
- `!lock` / `!unlock` - Закрыть/открыть канал

### Основные
- `!ping` - Проверить бота
- `!serverinfo` - Информация о сервере
- `!userinfo` - Информация о пользователе
- `!ticket [категория]` - Создать тикет
- `!cmds` - Список команд

### Кастомные команды
- `!addcommand <триггер> <ответ>` - Создать команду
- `!delcommand <триггер>` - Удалить команду
- `!mycommands` - Список команд

## 📝 Переменные для приветствия

- `{user}` - Упоминание пользователя
- `{username}` - Имя пользователя
- `{server}` - Название сервера
- `{member_count}` - Количество участников

## ⚠️ Требуемые права бота

- Administrator
- Manage Channels
- Manage Roles
- Ban Members
- Kick Members
- Manage Messages
