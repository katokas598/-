# Discord Bot + Web Dashboard

Мощный Discord бот с веб-панелью управления для модерации и настройки.

## 📦 Установка

### 1. Скачайте репозиторий

### 2. Запустите настройку

```bash
python setup.py
```

Введите данные:
- Токен Discord бота
- Токен Telegram бота
- Префикс команд
- ID Discord сервера
- Категории тикетов
- Логин и пароль для сайта (или оставьте пустыми для автогенерации)

### 3. Установите зависимости

```bash
cd bot
pip install -r requirements.txt

cd ../website
pip install -r requirements.txt
```

### 4. Запуск

**Вариант 1 - Всё вместе:**
```bash
python run.py
```

**Вариант 2 - Раздельно:**

Терминал 1 - Бот:
```bash
cd bot
python main.py
```

Терминал 2 - Сайт:
```bash
cd website
python -m uvicorn app:app --host 0.0.0.0 --port 8000
```

## 🌐 Доступ к сайту

После запуска сайт будет доступен по адресу:
- Локально: `http://localhost:8000`
- Через интернет: `http://ВАШ_IP:8000`

## 🌩️ Настройка HTTPS (Cloudflare)

Для безопасного доступа по HTTPS:

1. **Зарегистрируйтесь** на [cloudflare.com](https://cloudflare.com)

2. **Добавьте домен** `zyc-discord.duckdns.org`

3. **Измените NS сервера** в панели DuckDNS на Cloudflare

4. **Добавьте DNS запись** в Cloudflare:
   - Type: `CNAME`
   - Name: `zyc-discord`
   - Value: `zyc-discord.duckdns.org`
   - Proxy: `Proxied` (оранжевая иконка)

5. **Готово!** Теперь сайт доступен по:
   - `https://zyc-discord.duckdns.org`

## 🔧 Функционал бота

### Команды модерации
- `!ban @user [причина]` - Забанить пользователя
- `!unban ID` - Разбанить пользователя
- `!kick @user [причина]` - Кикнуть пользователя
- `!mute @user [время]` - Замутить пользователя
- `!unmute @user` - Размутить пользователя
- `!warn @user [причина]` - Выдать предупреждение
- `!warns @user` - Показать предупреждения
- `!clearwarns @user` - Очистить предупреждения
- `!clear [кол-во]` - Очистить сообщения
- `!lock` / `!unlock` - Закрыть/открыть канал

### Основные команды
- `!ping` - Проверить бота
- `!serverinfo` - Информация о сервере
- `!userinfo` - Информация о пользователе
- `!ticket [категория]` - Создать тикет
- `!cmds` - Список команд

### Кастомные команды
- `!addcommand <триггер> <ответ>` - Создать команду
- `!delcommand <триггер>` - Удалить команду
- `!mycommands` - Список ваших команд

## 🎨 Функционал сайта

| Страница | Описание |
|----------|-----------|
| Dashboard | Статистика сервера |
| Команды | Управление кастомными командами |
| Приветствие | Настройка приветствия новых участников |
| Тикеты | Просмотр тикетов |
| Настройки | Настройка префикса, категорий тикетов |

## 📝 Переменные для приветствия

- `{user}` - Упоминание пользователя
- `{username}` - Имя пользователя
- `{server}` - Название сервера
- `{member_count}` - Количество участников

## 🔐 Безопасность

- Пароли хешируются с помощью bcrypt
- Сессии хранятся в безопасных cookies
- HTTPS рекомендуется для продакшена
