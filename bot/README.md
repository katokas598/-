# DiscordSer Bot

Мультифункциональный бот для Discord с управлением через Telegram.

## Установка

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка бота

```bash
python setup.py
```

При настройке нужно указать:
- Токен Discord бота
- Токен Telegram бота
- Префикс команд (по умолчанию `!`)
- Telegram ID администраторов (через запятую)
- ID Discord сервера
- Категории тикетов

### 3. Запуск бота

```bash
python main.py
```

## Команды Discord

### Основные команды
- `!ping` - Проверить бота
- `!serverinfo` - Информация о сервере
- `!userinfo` - Информация о пользователе
- `!invite` - Приглашение на сервер
- `!help` - Список команд

### Модерация (администраторы)
- `!ban <@user> [причина]` - Забанить пользователя
- `!unban <id>` - Разбанить пользователя
- `!kick <@user> [причина]` - Кикнуть пользователя
- `!mute <@user> [время] [причина]` - Замутить (время: 10m, 1h, 1d)
- `!unmute <@user>` - Размутить пользователя
- `!warn <@user> [причина]` - Выдать предупреждение
- `!warns <@user>` - Показать предупреждения
- `!clear <количество>` - Очистить сообщения
- `!lock` - Заблокировать канал
- `!unlock` - Разблокировать канал

### Тикеты
- `!ticket [категория]` - Создать тикет
- `!closeticket` - Закрыть тикет (модераторы)

## Команды Telegram

После запуска используйте кнопки в меню:
- 📊 **Статистика** - Показать статистику модерации
- ⚙️ **Настройки** - Настроить роли и категории тикетов
- 🎫 **Тикеты** - Управление тикетами
- 📝 **Логи модерации** - Просмотр логов
- 🗑️ **Очистить логи** - Удалить все логи

## Требования

- Python 3.9+
- Ubuntu (рекомендуется) или Windows
- Токены Discord и Telegram ботов

## Получение токенов

### Discord бот
1. Перейдите на https://discord.com/developers/applications
2. Создайте новое приложение
3. Перейдите в Bot и создайте бота
4. Скопируйте токен
5. В OAuth2 выберитеScopes: bot
6. Выберите права: Administrator или нужные права

### Telegram бот
1. Начните диалог с @BotFather в Telegram
2. Используйте /newbot для создания бота
3. Скопируйте токен
